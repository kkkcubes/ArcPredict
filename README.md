# ArcPredict

AI-Powered Real-Time Prediction Markets on Arc

ArcPredict is a fully on-chain prediction market platform built on Arc Network.

Features:

- Create prediction markets
- Trade YES / NO positions
- Real-time blockchain event streaming
- Arc USDC settlement
- Portfolio tracking
- Analytics dashboard
- MCP AI assistant
- Arc App Kit integration
- Unified Balances
- Verification Center
